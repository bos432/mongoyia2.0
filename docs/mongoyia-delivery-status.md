## Test Acceptance Gate

A test server is acceptable only after this command passes:

```bash
php yii mongoyia-acceptance/run \
  --baseUrl=https://<test-domain> \
  --profile=test \
  --strict=1 \
  --cleanupAfterRun=1 \
  --interactive=0
```

Equivalent wrappers:

```powershell
.\console\shell\mongoyia-acceptance.ps1 -BaseUrl "https://<test-domain>" -Profile test -Strict -CleanupAfterRun -ImUrl "wss://<test-domain>/<im-path>"
```

```bash
PROFILE=test STRICT=1 CLEANUP_AFTER_RUN=1 BASE_URL=https://<test-domain> IM_URL=wss://<test-domain>/<im-path> sh console/shell/mongoyia-acceptance.sh
```

Expected result:

- Deployment check: `0 failure(s), 0 warning(s)`.
- Security scan: `0 failure(s), 0 warning(s)`.
- Data readiness: `0 failure(s), 0 warning(s)`.
- IM, API, frontend, backend, and payment regressions pass.
- Generated test data is cleaned after the run.
- Web closure cleanup verification returns zero pending `WEBFIX-*` rows.
