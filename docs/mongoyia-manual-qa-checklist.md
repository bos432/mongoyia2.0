# Mongoyia 手工 QA 清单

## 使用前提

手工 QA 前先跑自动化验收：

```bash
php yii mongoyia-acceptance/run \
  --baseUrl=https://<测试域名> \
  --profile=test \
  --strict=1 \
  --cleanupAfterRun=1 \
  --interactive=0
```

只有自动化验收通过后，才开始下面的人工抽查。
